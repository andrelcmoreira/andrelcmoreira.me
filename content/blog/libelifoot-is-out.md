Last week, the [first version](https://pypi.org/project/libelifoot/) of `libelifoot` was released. For those who don't know, `libelifoot` is a library for parsing and analyzing [Elifoot 98](https://www.elifoot98.com.br/) equipas (or "teams", in english). This library emerged from my own need to understand more (from a technical perspective) about the game that I spent so many hours playing in my childhood, and I hope it can be useful for other fans of the game as well. Through this library, you can easily dump informations about the equipa files as well as update them with upstream data, collected from public football data providers.

Furthermore I hope to write a more detailed post about the process of reverse engineering these files, but for now, you can browse more details [here](https://github.com/andrelcmoreira/libelifoot).

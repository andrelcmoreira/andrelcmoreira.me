libelifoot: version 0.1.3 is out


Mar 15, 2026 - 1 min

Last week, the first version of libelifoot was released. For those who don't know what I'm talking about, libelifoot is a library for parsing and analyzing Elifoot 98 equipas ("teams", in english). This library emerged from my own need, as a programmer, to understand more (from a technical perspective) about the game that I spent so many hours playing in my childhood. Through this library, you can easily dump informations about the equipa files as well as update them with upstream data, collected from public football data providers. I hope it can be useful for other fans of the game as well.

Furthermore I expect to write a more comprehensive post about the process of reverse engineering these files, but for now, you can get more details here.
